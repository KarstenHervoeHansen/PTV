Version2: 
this version is used for PCB before 20/10/2004. This version can cause the PT660M and PT660LS to errorneously change preset without clicking the buttons.

Version3: Feb,6 2006
this version is used for PCB which mark:  date 20/10/2004, Rev 1.0.  Version 3 has an extra connection from LCD_CLK to CPLD. This 27MHz clock allows the CPLD to make a filter on SCL and SDA signals. The filter corrected the bug that PT600LS changes preset when it is running for a few hours.
