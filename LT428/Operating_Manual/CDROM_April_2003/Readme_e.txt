Operating environment
  Pentium 100MHz or better
  Operating software Windows95 or WindowsNT4.0 or upper version.
  Use a color monitor to display 256 colors or more.
  Resolution of a monitor is 640 by 480 dots or more.
  Need a CD-ROM driver.

--------------------------------------------------------------------------
A way to connect between LT428 and a PC
  Please read the instruction manual paragraph 5.3 How to get started.

--------------------------------------------------------------------------
Instruction manual
  The instruction manual is contained in this CD-ROM by PDF file format.
  Acrobat Reader is required when you read the instruction manual. If the
  Acrobat Reader is not installed, please install Acrobat Reader to your
  PC by following method.

--------------------------------------------------------------------------
Installation of Acrobat Reader
1. Start PC. Insert this CD-ROM into a CD-ROM drive.
2. Open the folder of "English" from this CD-ROM by explorer.
3. Double click "ar405eng.exe" to install Acrobat Reader. Then follow the
   messages on the screen.
Note: The Acrobat Reader is the product of Adobe Systems Incorporated.
      Please read the attached file "Readme.txt" and "Grant of use of
      Attached Software" that is displayed on the screen when you install
      it.
