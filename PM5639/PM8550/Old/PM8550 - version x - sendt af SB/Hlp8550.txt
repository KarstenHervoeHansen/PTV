.id readstatus
.yupper 5
.title ' Help:  Read Status '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 The status of the color sensor is read into the computer
 and displayed in the Status field:

 NB: During start up of the calibration program
     the status of the color sensor is read and
     shown in the Status field.

^A34Serial Number          : KUxxxxxx^A30
 The serial number of the attached color sensor.

^A34Calibration Status     : FACTORY / USER^A30
 The active calibration of the color sensor. This
 may be either the original factory or the last
 previously performed by the user.

^A34Write Protect Status   : ON /OFF^A30
 The calibration data area in the color sensor
 may be protected against accidental destruction.
 This protection has to be removed before a the
 recalibration can be performed. Some color
 sensors may need to be disasembled to remove the
 write protection.
 Instructions for the actual head are obtained by
 selection of "Write Protect" in the main menu.

^A34User Text              :^A30
 A 10 character string which the user may use as a
 notepad for free information. The string is empty
 from factory.

.id writeprotect
.yupper 5
.title ' Help:  Write Protect '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 The write protect function is for protection of stored data
 in the color sensor. The protection may be implemented
 different in heads from different series. The instructions
 you get when you call the "Write Protect ON" or
 "Write Protect OFF" depends upon the serial number of the
 actual connected color sensor.


 To set write protection ON or OFF you may need the
 following tools:

   a)  2.5 mm Allen key
   b)  soldering iron
   c)  scalpel

 Please follow the instructions in the submenu
 "Write Protect ON" or "Write Protect OFF".

.id restore
.yupper 5
.title ' Help:  Restore '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index


 This function is used to restore the original
 factory calibration of the color sensor. When
 this function is performed any previously stored
 user calibration is overwritten and lost.

 NB. Before overwriting the user calibration you
     will be prompted to confirm it.

 After restoring the factory calibration the color
 sensor status is set to "FACTORY".

.id howtouse
.yupper 5
.ylower 22
.title ' Help:  How to use . . .'
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 How to use the PM8550 calibration software:

 1) Select the "Calibration  Setup" menu:
    - Select the chromaticity unit (optional) -
                                      default: CIE 1931
    - Select the luminance unit (optional) -
                                      default: cd/mý

 2) Enter the chromaticity coordinates (optional).

       If only the luminance value is to be calibrated
       the chromaticity coordinates may be omitted.

 3) Enter the luminance value.

       The luminance value must always be specified
       for the calibration.

 4) Enter User Text (optional).

       A user text may be stored in the color sensor.
       The user text-string can be used as a notepad for
       information. The user text is only overwritten
       when a new text string is specified. If you want
       to delete all the user text, please enter a
       "space" character. The text string is empty from
       factory.

 5) Start the calibration with the menu "Start Cal" or press
    the S-key.

.id lumcieunit
.yupper 5
.title ' Help:  Luminance / CIE Unit'
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 ^A34CIE:^A30

 The chromaticity coordinates of the reference
 monitor may be entered in three different
 formats. The range of valid values are:

     Unit:                            Value:
     xy     CIE 1931 coordinates      0.100 - 0.600
     u'v'   CIE 1976 coordinates      0.100 - 0.600
     uv     CIE 1960 coordinates      0.100 - 0.600

 ^A34Luminance:^A30

 The luminance you specify is the luminance which
 the color sensor will show on the monitor used
 for calibration. The range of valid values are:

     Unit:          Value:
     cd/mý          10  - 1000
     NIT            10  - 1000
     ftL            2.9 - 291

.title ' Help:  User Text '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 A user text may be stored in the color sensor.
 The user text-string can be used as a notepad for
 information. The user text is only overwritten
 when a new text string is specified or the factory
 calibration is restored.

 If you want to delete the user text, please enter
 a "space" character.

 The text string is empty from factory.

 A maximum of 10 characters may be entered.

.id startcal
.yupper 5
.ylower 22
.title ' Help:  Starting The Calibration '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 Before the start of the calibration, the
 calibration data must be entered within the valid
 limits.

 It is possible to calibrate the luminance only.
 The calibration of the chromaticity coordinates
 without calibrating the luminance is not possible.

 When the calibration is completed the status of
 the color sensor is set to "USER CAL".

        After a successful calibration, you should set
        Write Protect ON with the menu "Write Protect".

    NB. It is possible to use the color sensor when
        the Write Protect is OFF, but it is recommended
        to set it ON to protect the calibration data in
        the color sensor.
