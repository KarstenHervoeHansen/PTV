.id readstatus
.yupper 5
.title ' Help:  Read Status '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 The status of the color sensor is read and then
 shown in the Status field:

    Serial Number
    Calibration Status
    Write Protect Status
    User Text

 NB: During start up of the calibration program
     the status of the color sensor is read and
     shown in the Status field

.id writeprotect
.yupper 5
.title ' Help:  Write Protect '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 To set write protect ON or OFF you need the
 following tools:

   a)  2.5 mm unbrako screwdriver
   b)  soldering iron
   c)  scalpel

 Then please follow the instructions in the submenu
 "Write protect ON" or "Write protect OFF"

.id restore
.yupper 5
.title ' Help:  Restore '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 If the color sensor status is "USER" then the
 factory calibration is restored and the user
 calibration is overwritten.

 NB. Before overwriting the user calibration you
     will be prompted to confirm it.

 After restoring the factory calibration the color
 sensor status is set to "FACTORY".

.id howtouse
.yupper 4
.ylower 22
.title ' Help:  How to use... '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 How to use the PM8550 calibration software:

 1) Select the "Calibration  Setup" menu:
    - Select the chromaticity unit (optional) -
                                      default: CIE 1931
    - Select the luminance unit (optional) -
                                      default: cd/mý

 2) Enter the chromaticity coordinates (optional)

 3) Enter the luminance value

 4) Enter User Text (optional)

 5) Start the calibration with the menu "Start Cal" or press
    the S-key

.id startcal
.yupper 5
.title ' Help:  Starting The Calibration '
 ^A3EEscape: Exit Help^A30                          ^xIndex of Help^x
.cref index

 Before the start of the calibration, the calibration
 data must be entered within the valid limits.

 NB. It is possible to calibrate the luminance only, but it
     is not possible to calibrate the cromaticity coordinates
     without calibrating the luminance.

 After a succesfull calibration, the status of
 the color sensor is set to "USER CAL".
 After a succesfull calibration, you should set
 Write Protect ON with the menu "Write Protect"

 NB. It is possible to use the color sensor when
     the Write Protect is OFF, but it is recommended
     to set it ON to protect the calibration data in
     the color sensor
