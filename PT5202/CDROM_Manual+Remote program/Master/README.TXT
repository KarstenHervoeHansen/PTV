Welcome to the DK-Technologies's Installation CD for the PT5202 Windows(R) Interface.

Content of this CDROM:

	- Adobe Acrobat Reader
	- PT5202 User's manual
	- PT5202 Data sheet
	- PT5202 Windows application program (PT5202remote.exe).
	- Help file for applikation program. (PT5202.hlp).


To run the s/w for the PT5202:

	* Copy the file PT5202remote.exe to your harddisk.
	* Copy the help file PT5202.hlp to your harddisk.
	* If run directly from the CDROM, no INI file can be stored.

To read the .pdf files:
You need Adobe Acrobat Reader (R) installed on your system. 
You may install the Reader from this CD by running the ar405eng.exe file.
