Create and Modify PC Board Outlines
Duane Thompson has submitted a paper outlining a method of using "IntelliCAD" (replacement for "VisualCAD") to create an outline, export a DXF file, and then import that DXF file into Layout. Back-annotation of the Capture schematic from the Layout .MAX pc board file stuffs the pcb footprint location and rotation into a part field in the schematic, which then allows creation of a netlist which contains footprint location information.

Netlist interfaces
Ole Ejlersen of Nordcad Systems, A/S, a Cadence VAR from Denmark has contributed two netlist interfaces made for the DDE (SuperMax/Dansk/IPL) system. Be sure to read the installation notes also included.

Netlist interface to DDE Supermax (IPL/Dansk) for Capture and Capture CIS. This is a very basic and simple netlist formatter for the DDE system. This netlist extracts net and part information into a DDE format. You will find the netlist in the other tab of the create netlist dialog and it is called iwl.

Netlist interface to DDE Supermax (IPL/Dansk) for Capture and Capture CIS. This is a much more comprehensive netlist formatter for the DDE system. The current version is developed for Release 9.1 of Capture and Capture CIS but will work fine with 9.2.x. The installation will also ask the user to overwrite some files - make sure to say NO to this. This netlist extracts net and part information into a DDE format plus many parameters that are understood by the IPL PCB system such as trackwidth, impedance etc. You will find the netlist in the other tab of the create netlist dialog and it is called IPL.EXE.

Padstacks
Padstacks are the basic building block from which all your Component Footprints and OrCAD Layout PCB files are developed. It is important to understand the make-up of the padstack, how to change, edit, rename & save them and even to spend some time setting up your own site specific padstacks. Mike Montaigne has submitted this technical note in four files:
How to open a Padstack library
Starting Padstacks Spreadsheet File
It is much easier to start with a spreadsheet 'big picture' first. You can even put in the 'rules' (Plane = Drill + 30, Top, Bot, Inner = Drill + 20, Solder Mask = Top, Bot + 10 etc.), but I found this only a starting point and I modified the spreadsheet significantly before actually making my Padstack.
Starting Padstacks Layout Library
Started with Padstack.LLB, made my own copy and then added in my spreadsheet site specific pads and some new ones, as I went along.
