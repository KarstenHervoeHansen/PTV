Welcome to the DK-Audio's Installation CD for the PT5201 Windows(R) Interface.



To run the s/w for the LT428:

	* Copy the file LT428 remote.exe to your harddisk.
	* Copy the help file LT428.hlp to your harddisk.
	* If run directly from the CDROM, no INI file can be stored.

To read the .pdf files:
You need Adobe Acrobat Reader (R) installed on your system. 
You may install the Reader from this CD by running the ar405eng.exe file. 

Please register the software!

After Adobe Acrobat Reader (R) is installed, click the INTRODUCTION.PDF file on the CD.
