Changes in Version 1.2:
Settings of Status bar and System Status is now saved in the Windows Registry.
Leaving a settings window with cancel willno longer generate the question; "Close window and loose all changes?" if none of the settings have been changed.

Support for help file.


Changes in Version 1.01:
In TSG pattern selection; the pattern "Multiburst" is no longer supported. When upgrading to version 1.01 please ensure that the TSG pattern selection in the PT5201 Compact VariTime Sync Generator is different from "Multiburst". This also applies to the four presets.
