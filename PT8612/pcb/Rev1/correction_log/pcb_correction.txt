
Fejl konstateret på PT8612 rev0 :

	-- Konnektor J3 pin21 og pin26 er kortsluttet på FPGA'en pin AA17.

	-- annullar ring 0.2m for lille ( forskellen mellem via diameter og via hole ).
	   Mindste annular ring Elprint kan producere er 0.3, hvilket samtidig forøger board pris.
	   Løsning : Lav alle via huller under FPGA'en mindre og diameteren en smule større, så Elprints
	   krav til mindste annular ring er overholdt. Og resten af vias på boardet kan laves med ikke fordyrende
 	   dimensioner.
	   ( 21 mil via diameter og 9 mil hole overholder Elprint's krav. )

	-- vref spænding, der indikerer grænsen mellem high og low på IC17 og IC18 burst ram, er ikke defineret.
	
	-- pullups til Configuration PROM IC23 ( R48, R49, R50 ) er ikke forbundet til pullup voltage VIO33.
	   
	-- CQ1\ er ikke forbundet fra RAM (IC17) til FPGA
